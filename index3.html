Úvod

Cascading Style Sheets (CSS), česky kaskádové styly, je stylový jazyk sloužící pro popis vzhledu a formátování dokumentů napsaných nějakým značkovacím jazykem. Odděluje styl dokumentu od jeho obsahu. Nejčastěji se používá pro přidávání stylů (např. fonty, barvy, pozadí, ohraničení, formátování textu) do webových dokumentů. Kaskádové styly umožňují definovat způsob zobrazení každého elementu na stránce. CSS lze propojit s HTML dokumentem jako vnitřní, externí nebo vložené styly.

Možnosti propojení s HTML dokumentem

Vnitřní styly

Metoda propojení prostřednictvím vnitřních stylů spočívá v uvedení CSS kódu do hlavičky HTML souboru, který chceme stylovat pomocí CSS. Každý HTML soubor tak bude obsahovat CSS kód ke stylování dané stránky. Tato metoda je vhodná jen pokud chceme stylovat jedním stylem pouze jednu stránku. Kdybychom totiž chtěli provést jednu a tu samou změnu na více stránkách, museli bychom změnit CSS kód v každém HTML souboru zvlášť.

Externí styly

Princip externích stylů spočívá ve vytvoření samostatného souboru pouze pro CSS. Tento soubor má příponu css a Ize vytvořit v jakémkoliv textovém nebo HTML editoru. Propojení s HTML soubory se docílí vložením odkazu na CSS soubor do hlavičky každého z HTML souborů, který má být daným CSS souborem stylován. Všechny HTML soubory se tak odkazují na jeden soubor CSS. To znamená, že když chcete provést změnu v zobrazení všech stránek, stačí upravit jediný CSS soubor. Tím je vyřešen problém předchozí metody.

Vložené styly

Poslední možností jsou vložené styly, které stylují elementy prostřednictvím atributu style umístěného přímo v HTML tagu elementu, který mají stylovat.
